Vuepress

Vue 作者尤雨溪于 2018 年 4 月 13 日推出基于 Vue、围绕 markdown 生成静态页面的 VuePress 框架。

用途

1、快速搭建一个个人博客

2、帮助专注写一个好看的文档，关注内容，样式简约

特性

  • 内置 markdown 扩展，针对技术文档进行了优化
  • 能够利用内嵌在 markdown 文件中的 Vue 代码
  • 以 Vue 驱动的自定义主题系统
  • PWA 支持
  • Google Analytics 集成
  • 内置强大插件
